# Lesson Instructions

## 1. Development Setup

Before you start, you'll need two things installed on your computer: **Python** and **Visual Studio Code (VS Code)**.

### Install Python

1. Download Python from the official site: [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Run the installer.
   - **Windows:** Make sure to check the box that says **"Add Python to PATH"** before clicking Install.
   - **Mac:** Follow the installer prompts (you may need Xcode Command Line Tools, which the installer will prompt for if needed).
3. Verify the installation by opening a terminal (Command Prompt, PowerShell, or Terminal) and running:
   ```bash
   python --version
   ```
   You should see something like `Python 3.12.x`.

### Install VS Code

1. Download VS Code from the official site: [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Run the installer and follow the setup prompts (defaults are fine).
3. Once installed, open VS Code and install the **Python extension**:
   - Click the Extensions icon in the sidebar (or press `Ctrl+Shift+X` / `Cmd+Shift+X`)
   - Search for "Python" (by Microsoft) and click **Install**

## 2. Hello World Activity

Let's make sure everything works by writing and running your first Python program.

1. Open VS Code.
2. Create a new folder for this project (e.g., `development_setup`) and open it in VS Code (`File > Open Folder`).
3. Inside that folder, create a new file called `1st_exercise.py`.
4. Add the following code to `1st_exercise.py`:
   ```python
   print("Hello, World!")
   ```
5. Run the file:
   - Open a terminal in VS Code (`Terminal > New Terminal`)
   - Run:
     ```bash
     python hello.py
     ```
6. You should see this output in the terminal:
   ```
   Hello, World!
   ```

If you see that message, your development environment is set up correctly! 🎉
